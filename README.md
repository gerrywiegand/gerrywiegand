## 🚀 Featured Projects

### [Pokemon Stat Analyzer](https://github.com/gerrywiegand/Pokemon-Stat-analyzer)
A competitive Pokemon reference tool built with React and Vite. Search any Pokémon by name or Dex number and get base stats with visualization bars, a full 18×18 type matchup matrix, complete move lists by category, ability descriptions, and a persistent favorites system.

**Tech used:** React, Vite, JavaScript, PokéAPI
[Live Demo](https://pokemon-stat-analyzer.onrender.com) | [Repo](https://github.com/gerrywiegand/Pokemon-Stat-analyzer)

---

### [CatchLog](https://github.com/gerrywiegand/CatchLog)
A full-stack fishing log app built for use in the field — mobile-friendly with large touch targets designed for wet hands. Users can authenticate, log catch data (species, weight, length, lure), manage a species database, and paginate through their catch history.

**Tech used:** React, Flask, SQLAlchemy, SQLite, Material UI, Python
[Demo](https://www.loom.com/share/17635d25f7ae4d71ba356157d2c45a2f) | [Repo](https://github.com/gerrywiegand/CatchLog)

---

### [GardenGrid](https://github.com/gerrywiegand/GardenGrid)
A full-stack garden planning app with a grid-based bed layout. Users build a personal plant library, place plants on coordinate grids, and define companion-planting relationships that show live visual warnings (green = beneficial, red = detrimental) when placing plants near each other.

**Tech used:** React, Flask, Flask-JWT-Extended, SQLAlchemy, Python
[Demo](https://www.loom.com/share/f85ddb1606ed475f9458e3c829eba91c) | [Repo](https://github.com/gerrywiegand/GardenGrid)
